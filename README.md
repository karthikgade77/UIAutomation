# UIAutomation
Emirates UI Automation for the Scenario with Cucumber

Scenario: Bill wants to travel from DXB to LHR
Given that Bill has decided to check available flights
When he looks at a return trip from DXB to LHR leaving one week from now
Then he should be shown the cheapest return ticket from DXB to LHR

Cucumber project can be executed with any of the following: Feature File / Runner Class / POM.xml

Geckodriver.exe available in Resource folder to support execution via Mozilla Firefox
